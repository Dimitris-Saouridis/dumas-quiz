# 📚 Alexandre Dumas Quiz Game

Welcome to my quiz project! 🎮

This is a Python quiz game based on three classic novels by the famous author **Alexandre Dumas**:

- *The Count of Monte Cristo*
- *The Three Musketeers*
- *The Black Tulip*

---

## 🧠 About the Game

In this quiz, players answer multiple-choice questions about the stories, characters, and events from these novels.

At the end, the score is calculated based on correct answers.

---

## 🚀 Features

- Multiple-choice questions (A, B, C, D)
- Instant feedback (Correct / Incorrect)
- Final score percentage
- Based on classic literature

---

## 📖 Author

All questions are based on works by **Alexandre Dumas**.
